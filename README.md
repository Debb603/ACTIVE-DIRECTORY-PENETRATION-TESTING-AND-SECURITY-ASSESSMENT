# ACTIVE-DIRECTORY-PENETRATION-TESTING-AND-SECURITY-ASSESSMENT
This project focuses on Active Directory penetration testing and security assessment. It involves evaluating the security posture of an Active Directory environment by identifying vulnerabilities, testing security controls, and demonstrating common attack techniques in a controlled lab environment.

Tools used
Nmap         -             Network discovery and port scanning.
NetExec (NXC) -            SMB, LDAP, and authentication enumeration; credential validation; and password spraying activities.
Kerbrute      -            Validation and enumeration of Active Directory usernames through Kerberos.
AD Username Generator  -   Generation of potential usernames based on employee naming conventions.
Windows Command Prompt  -  Execution of domain enumeration commands such as net user /domain and net accounts /domain.
PowerShell        -        Collection of system and domain information during the assessment.
Kali Linux        -        Penetration testing platform used to perform enumeration and authentication testing activities.
